# Contributing

We welcome all contributions to improve this project! This includes:

- Bug reports
- Bug fixes
- Documentation improvements
- Enhancements
- Ideas and suggestions

## How to Contribute

1. **Fork the repository** – Create a personal copy of the repository.
2. **Create a branch** – Create a new branch for your feature or fix.
3. **Make changes** – Make your changes in the new branch.
4. **Commit changes** – Commit your changes with a clear message.
5. **Push your branch** – Push your branch to your forked repository.
6. **Create a Pull Request** – Open a Pull Request from your branch to the main repository.

Thank you for contributing to this project! ❤️
