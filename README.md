# Simple Interest Calculator

This is a simple Python program that calculates the Simple Interest based on the given principal, rate of interest, and time period.

## How to use:
1. Clone this repository to your local machine.
2. Run the Python script `simple_interest_calculator.py`.
3. Provide the input values for the principal amount, rate of interest, and time period.
4. The program will output the calculated simple interest.

## Formula Used:
Simple Interest = (Principal * Rate * Time) / 100
